== README

Requirements:

* Ruby 2.3.3

* Rails 4.2.7.1

* PostgreSQL 9.6.3

After cloning the repo, run the following to get started:

This will install any missing Gems and update any Gems you already have installed
* <tt>bundle install</tt>

This will create the database
* <tt>rake db:create</tt>

This will set up the tables needed for this app
* <tt>rake db:migrate</tt>
